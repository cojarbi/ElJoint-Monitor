# Summary of Media Documents

This document summarizes the contents and structure of two Excel files related to the "Día de la Madre 2025" campaign for **El Fuerte**.

---

## 1. Presupuesto El Fuerte Dia de la Madre 2025.xls
**Purpose:** Media buying budget and plan for the campaign.

### Tabs:
| Tab | Description |
|-----|-------------|
| **MEDCOM** | Detailed media schedule for Telemetro. Includes programs, broadcasting days, time slots, and spot durations (35ss, 10ss). |
| **TVN** | Detailed media schedule for TVN-2, following the same structure as MEDCOM. |
| **Resumen** | Financial summary showing totals per channel ($11,652.56 for Telemetro, $10,214.91 for TVN) and the grand total including ITBMS. |

### Key Fields:
- **Programa:** Name of the TV show.
- **Días:** Scheduled days for broadcast (e.g., L-V, D-M).
- **Horario:** Airing time window.
- **Duración:** Duration of the spot (e.g., 35 seconds).
- **Tarifa neta:** Net cost per spot.

---

## 2. monitoreo del fuerte.xls
**Purpose:** Monitoring log from Kantar Ibope Media showing actual aired spots.

### Tabs:
| Tab | Description |
|-----|-------------|
| **Consulta Infoanalisis** | The primary data sheet containing 854 records of aired spots. |
| **Datos de Consulta** | Metadata about the export from the Kantar system. |

### Key Data Fields:
- **Anunciante:** TAMEK SA
- **Marca:** EL FUERTE
- **Medio:** Media outlet (e.g., TELEVISION NACIONAL)
- **Vehiculo:** Specific channel (TVN, Telemetro, etc.)
- **FECHA:** Date aired (YYYYMMDD)
- **Duración_Real:** Actual seconds aired.
- **Soporte:** Program name where the ad appeared.
- **Total_Inversion:** Financial cost attributed to the aired spot.

---

## Summary Conclusion
The **Presupuesto** file represents the "Plan" (What was purchased), while the **Monitoreo** file represents the "Execution" (What actually aired). These files are typically compared to ensure that the media company delivered exactly what was paid for.
